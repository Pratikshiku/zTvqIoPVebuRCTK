## Tourism Information Management System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/5153bb3025b74917b29bd4a96d20b0a6/ghb20250919) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (any of the listed software can be used)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Functional Description
The system is divided into (roles): regular users, administrators.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Travel Package Ordering
5. Travel Information
6. Personal Center
7. Travel Purchase
8. My Favorites


**Administrator Functions:**
1. Login
2. Personal Center
3. User Management
4. Travel Package Management
5. Travel Purchase Management
6. Travel Information Management
7. Carousel Management
